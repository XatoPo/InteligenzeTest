Documentación de la API

1. Insertar Datos

Descripción: Inserta datos en la base de datos. Este endpoint obtiene datos de diversas fuentes y los inserta en la base de datos.

URL: /insert-data

Método: POST

Headers:

Content-Type: application/json

Ejemplo de Cuerpo:

            {
    "filteredAsteroids": [/* Datos de asteroides */],
    "filteredMarsPhotos": [/* Datos de fotos del rover */],
    "dailyAstronomyData": [/* Datos de astronomía diaria */],
    "epicImagesData": [/* Datos de imágenes EPIC */]
}

Respuesta:

{
    "message": "Datos insertados correctamente"
}

2. Obtener Imágenes del Día (APOD)

Descripción: Obtiene datos de la Imagen Astronómica del Día (APOD) desde una API externa.

URL: /apod

Método: GET

Respuesta:

{
    "date": "2024-07-15",
    "explanation": "Descripción de la imagen.",
    "url": "https://example.com/image.jpg"
}

3. Obtener Fotos de Rovers en Marte

Descripción: Obtiene fotos tomadas por los rovers en Marte desde una API externa.

URL: /mars-rover-photos

Método: GET

Respuesta:

[
    {
        "id": 1,
        "earth_date": "2024-07-14",
        "camera": "FHAZ",
        "img_src": "https://example.com/rover.jpg"
    },
    ...
]
